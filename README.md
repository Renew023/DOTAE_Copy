## 🎮 게임 소개

### 장르 및 구성

- 주요 장르: 2D, 오픈월드, RPG, 어드벤처, 로그라이크  
- 구성 요소:
  - 전투, 채집, 거래를 통해 캐릭터 성장 가능
  - 성장 후 강한 적과의 전투, 종족의 영웅 또는 적이 될 수 있음
  - 배고픔/목마름 시스템 존재: 생존 요소 강화
  - 마을과의 관계를 통해 평화로운 해결도 가능

### 트레일러 및 스크린샷

> ⚠️ 현재 스크린샷과 트레일러는 준비 중입니다. (예정: 2025.07.20)

| 구성 요소 | 설명 |
|-----------|------|
| 시작 화면 | 메인 메뉴 및 UI |
| 전투 장면 | 실시간 전투 및 스킬 사용 |
| NPC 퀘스트 | 상호작용 및 의뢰 |
| 채집 & 빌딩 | 자원 수집 및 건축 |
| 상점 & 아이템 | 경제 시스템, 전리품 획득 |

### 미리 해보기

1. GitHub 저장소 클론
2. Unity 2022.3.17f에서 `DontFixScene` → `StartScene` 실행
3. (예정) 빌드된 버전으로 접근 가능
